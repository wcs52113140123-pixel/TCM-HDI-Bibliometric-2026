# WoS Partial-Year 2026 Supplement

This directory holds the supplementary search for 2026 partial-year publications 
(January 1 - May 13, 2026).

## Purpose

This data is used **only** for Discussion-level commentary on emerging trends. 
It is **not** included in the main bibliometric analysis, because 2026 is an 
incomplete year as of the search date (2026-05-13).

## Data redistribution restrictions

Same restrictions as the main `../wos/` directory. See `../wos/README.md` for 
reproduction steps. The only difference is `PY=(2026-2026)` in place of 
`PY=(2005-2025)`.

## Expected output

1 file, 83 records, ~0.86 MB.
